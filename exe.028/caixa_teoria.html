Trabalhando com caixas

Elementos de caixas

Tipos de caixas

box-level

são tipos de caixas que ocupam a largura total da tela do navegador

por exemplo se eu tenho um um texto que vai ate aqui.
o box-level vai se iniciar em outra linha e ocupara essa linha por inteiro
e caso uma outra caixa seja criada ela se iniciara em uma outra linha

são exemplos de box-level as caixas/tag

  • div
  • h1-h6
  • main
  • header
  • video
  • img
  • nav
  • artifice
  • entre outros

inline-level

ja os elementos do tipo inline-level não ocupam um espaço todo.

por exemplo se eu quero criar um elemento do tipo inline-level essa caixa não vai ocupar a linha toda e caso outra caixa seja criada ela se iniciara (se huver espaço) na mesama linha da caixa inline-level

são exemplos de inline-level as caixas/tags

  • span
  • a
  • code
  • small
  • strong
  • sub-sup
  • button
  • label
  • entre outros
